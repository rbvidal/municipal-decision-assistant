# Google Stitch Export — Kommunale Entscheidungsplattform

## Purpose

This directory stores the **frozen Google Stitch UI export** for the Kommunale Entscheidungsplattform. These files are the approved visual baseline generated by Google AI Studio (Stitch) from the UX specification.

## Important Rules

- **Never edit these files manually.** They are a design reference, not source code.
- **These files are not the React application.** The React implementation will be built in a separate directory, referencing these screens as the visual target.
- **This directory is a permanent design reference.** It captures the approved UX at the point of Stitch generation. Future design changes must be approved against this baseline.

## Directory Structure

```
stitch-export/
├── login/              Login screen, forgot password, reset password
├── home/               Startseite — operational workbench
├── case-workspace/      Case workspace with all contextual tabs
├── knowledge/           Wissen — global knowledge repository
├── documents/           Dokumentenverwaltung — global document management
├── new-case/            New Case Wizard
├── supervisor/          Supervisor Approval Workspace
├── administration/      Verwaltung — Übersicht, audit, jobs
├── corpus/              Korpus-Status — corpus health, inventory
└── users/               User & Role Management
```

## Per-Screen Contents

Each screen directory contains:

| Directory | Contents |
|---|---|
| `html/` | Exported HTML files (read-only reference) |
| `css/` | Exported CSS files (read-only reference) |
| `screenshots/` | Screen captures of the generated UI |
| `assets/` | Exported image assets, icons, logos |
| `README.md` | Screen metadata — purpose, backend module, endpoints, export info |

## Export Information

- **Source:** Google AI Studio (Stitch)
- **Specification:** `docs/STITCH_SPECIFICATION.md`
- **Design System:** `docs/DESIGN.md`

- **Status:** Frozen — visual baseline for React implementation
